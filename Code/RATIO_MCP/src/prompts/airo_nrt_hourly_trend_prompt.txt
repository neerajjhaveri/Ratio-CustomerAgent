# HOURLY CROSS‑SERVICE IMPACT (WHEN USER ASKS: "other services", "other offerings", "hourly trend", "trend", "first hours", "early impact")
Trigger ONLY if user explicitly wants hourly trend or asks about OTHER services impacted.

SCHEMA ALIGNMENT / IMPORTANT NOTE
OfferingAirodTrendTest does NOT contain IncidentId. Hourly impact is inferred by:
1) Identifying impacted services for the Incident via Factless_IncidentImpactedServices (ResponsibleIncidentId = @IncidentId, IsQCService = 1).
2) Applying the Incident’s global time window (Fact_Outage.ImpactStartDate/-2h .. MitigateDate/+10h) to hourly rows in OfferingAirodTrendTest filtered by ServiceName (and Offering where present).

DATA SOURCES
- OfferingAirodTrendTest: hourly AIRO/AIR-O/D basis data (columns: CalendarDayHour, ServiceName, Offering, Region, CustomerName, AR_Fleet, AR_Region, AR_Customer, AR_Subscription, ImpactedResourceCount, ImpactDuration).
- Factless_IncidentImpactedServices: impacted services list (ResponsibleIncidentId alias IncidentId). (Schema lists ResponsibleIncidentId as DATETIME; treat as INT IncidentId. If type mismatch encountered, add explanation.)
- Fact_Outage: Incident-level ImpactStartDate & MitigateDate (time window anchor).

REQUESTED VS OTHER SERVICES
- Requested service: inferred explicit ServiceName / Offering from user text; always include if present in OfferingAirodTrendTest within window (even if zero impact).
- OTHER services: ImpactedService values (IsQCService = 1) for the Incident.
- MULTI-CRITERION selection for TOP-N:
  1. Compute per-service aggregates inside window:
     CumImpactedResource, CumAIRO = SUM(ImpactedResourceCount), SUM(ImpactedResourceCount)/NULLIF(SUM(AR_Fleet),0),
     PeakHourAIRO, PeakHourImpactedResource, PositiveImpactHours.
  2. Build three ranked lists (default K=5): Intensity (CumAIRO), Peak (PeakHourAIRO), Volume (CumImpactedResource).
  3. Union lists + requested service.
  4. Add high-ImpactScore services until cap (ImpactScore = 0.4*N_CumAIRO + 0.3*N_PeakAIRO + 0.2*N_CumImpRes + 0.1*N_PeakImpRes using min-max normalization).
  5. Auto-include services with PeakHourAIRO >= @PeakAIROIncludeThreshold (0.12 default).
- Defaults: @MaxOtherServices = 5 (hard cap 15 unless "all services" -> include all zero-impact, still cap total hourly rows at 500).
- User overrides:
  "volume only" -> use Volume list only.
  "intensity only" -> Intensity list only.
  "all services" -> bypass ranking (subject to row cap).
  Custom count: "top 12" sets @MaxOtherServices=12.
- Zero-impact services excluded unless user says "include zeros" / "full zero timeline" OR only the requested service is zero-impact (then include with explanation).

TIME WINDOW (FACT_OUTAGE ANCHORED)
1. Get ImpactStartDate, MitigateDate for @IncidentId.
2. BaseAnchorHour = ImpactStartDate.
3. StartHour = DATEADD(hour,-2, ImpactStartDate).
4. NominalEnd = DATEADD(hour,+10, ImpactStartDate).
5. EndHour = CASE WHEN MitigateDate IS NOT NULL AND MitigateDate < NominalEnd THEN MitigateDate ELSE NominalEnd END.
6. Filter OfferingAirodTrendTest rows where CalendarDayHour BETWEEN StartHour AND EndHour.
7. If ImpactStartDate NULL: ask for clarification or alternate anchor; otherwise no hourly section.
8. Extend beyond +10 up to +24 only if explicitly requested (never > +24 without confirmation).

INCLUSION RULES
- Always include requested service’s hours (apply zero filtering rules).
- Remove pure zero-impact hours for other services unless "include zeros".
- If requested service entirely zero: include and add explanation.
- If NO other QC services selected: add explanation.

OUTPUT OBJECTS (append after Fleet/Region/Customer slices)
Fields:
  ServiceName, Offering, CalendarDayHour, ActiveResourceFleetCount (MAX(AR_Fleet)),
  ImpactedResourceCount (SUM hour),
  AIRO_Hourly (DECIMAL(10,4)) = 1.0 * SUM(ImpactedResourceCount)/NULLIF(MAX(AR_Fleet),0),
Do NOT add Region / Customer unless user explicitly asks for region- or customer-level hourly trend (then separate objects with AIRO_Hourly_Region / AIRO_Hourly_Customer fields).

ORDER
Requested service rows first (chronological), then other services ordered by:
  1. ListAppearances DESC
  2. ImpactScore DESC
  3. ServiceName ASC
Each service’s hours in ascending CalendarDayHour.

SAMPLE QUERY SKELETON (AIRO)
-- Parameters:
-- @IncidentId INT
-- @RequestedServiceName NVARCHAR(NULL)
-- @MaxOtherServices INT = 8
-- @TopPerCriterion INT = 5
-- @PeakAIROIncludeThreshold DECIMAL(5,4) = 0.12
-- @IncludeZeros BIT = 0
-- @SelectionMode VARCHAR(NULL)  -- 'VOLUME','INTENSITY','ALL', NULL=multi

WITH OutageWindow AS (
  SELECT ImpactStartDate, MitigateDate
  FROM Fact_Outage
  WHERE IncidentId = @IncidentId
),
WindowBounds AS (
  SELECT
    ImpactStartDate AS AnchorHour,
    DATEADD(hour,-2, ImpactStartDate) AS StartHour,
    CASE WHEN MitigateDate IS NOT NULL AND MitigateDate < DATEADD(hour,10, ImpactStartDate)
         THEN MitigateDate
         ELSE DATEADD(hour,10, ImpactStartDate)
    END AS EndHour,
    CASE WHEN MitigateDate IS NOT NULL AND MitigateDate < ImpactStartDate THEN 1 ELSE 0 END AS MitigateBeforeStartAnomaly
  FROM OutageWindow
),
ImpactedServices AS (
  SELECT DISTINCT ImpactedService AS ServiceName
  FROM Factless_IncidentImpactedServices
  WHERE ResponsibleIncidentId = @IncidentId
    AND IsQCService = 1
    AND COALESCE(ImpactedService,'') <> ''
),
CandidateServices AS (
  SELECT ServiceName FROM ImpactedServices
  UNION ALL
  SELECT @RequestedServiceName WHERE @RequestedServiceName IS NOT NULL
),
HourlyBase AS (
  SELECT
    t.ServiceName,
    t.Offering,
    t.CalendarDayHour,
    MAX(t.AR_Fleet) AS AR_Fleet_Hour,
    SUM(t.ImpactedResourceCount) AS ImpactedResource_Hour
    --SUM(t.ImpactDuration) AS ImpactDuration_Hour
  FROM OfferingAirodTrendTest t
  CROSS JOIN WindowBounds wb
  WHERE t.CalendarDayHour BETWEEN wb.StartHour AND wb.EndHour
    AND t.ServiceName IN (SELECT ServiceName FROM CandidateServices)
  GROUP BY t.ServiceName, t.Offering, t.CalendarDayHour
),
PerServiceAgg AS (
  SELECT
    ServiceName,
    Offering,
    SUM(ImpactedResource_Hour) AS CumImpactedResource,
    SUM(AR_Fleet_Hour) AS CumAR_Fleet,
    CASE WHEN SUM(AR_Fleet_Hour)>0
         THEN 1.0*SUM(ImpactedResource_Hour)/NULLIF(SUM(AR_Fleet_Hour),0) END AS CumAIRO,
    MAX(1.0*ImpactedResource_Hour/NULLIF(AR_Fleet_Hour,0)) AS PeakHourAIRO,
    MAX(ImpactedResource_Hour) AS PeakHourImpactedResource,
    SUM(CASE WHEN ImpactedResource_Hour>0 THEN 1 ELSE 0 END) AS PositiveImpactHours
  FROM HourlyBase
  GROUP BY ServiceName, Offering
),
Rank_Intensity AS (
  SELECT ServiceName,
         ROW_NUMBER() OVER (ORDER BY CumAIRO DESC, PeakHourAIRO DESC, ServiceName) AS rk
  FROM PerServiceAgg
),
Rank_Peak AS (
  SELECT ServiceName,
         ROW_NUMBER() OVER (ORDER BY PeakHourAIRO DESC, CumAIRO DESC, ServiceName) AS rk
  FROM PerServiceAgg
),
Rank_Volume AS (
  SELECT ServiceName,
         ROW_NUMBER() OVER (ORDER BY CumImpactedResource DESC, PeakHourImpactedResource DESC, ServiceName) AS rk
  FROM PerServiceAgg
),
TopLists AS (
  SELECT ServiceName,'INTENSITY' ListType FROM Rank_Intensity WHERE rk <= @TopPerCriterion
  UNION ALL
  SELECT ServiceName,'PEAK' FROM Rank_Peak WHERE rk <= @TopPerCriterion
  UNION ALL
  SELECT ServiceName,'VOLUME' FROM Rank_Volume WHERE rk <= @TopPerCriterion
),
Normalized AS (
  SELECT p.*,
         CumAIRO / NULLIF(MAX(CumAIRO) OVER(),0) AS N_CumAIRO,
         PeakHourAIRO / NULLIF(MAX(PeakHourAIRO) OVER(),0) AS N_PeakHourAIRO,
         CumImpactedResource / NULLIF(MAX(CumImpactedResource) OVER(),0) AS N_CumImpRes,
         PeakHourImpactedResource / NULLIF(MAX(PeakHourImpactedResource) OVER(),0) AS N_PeakImpRes
  FROM PerServiceAgg p
),
Scores AS (
  SELECT n.*,
         COALESCE(N_CumAIRO,0)*0.4
       + COALESCE(N_PeakHourAIRO,0)*0.3
       + COALESCE(N_CumImpRes,0)*0.2
       + COALESCE(N_PeakImpRes,0)*0.1 AS ImpactScore,
         (SELECT COUNT(DISTINCT ListType) FROM TopLists tl WHERE tl.ServiceName = n.ServiceName) AS ListAppearances
  FROM Normalized n
),
SeedSelected AS (
  SELECT DISTINCT ServiceName FROM TopLists
  UNION ALL
  SELECT ServiceName FROM Scores WHERE PeakHourAIRO >= @PeakAIROIncludeThreshold
  UNION ALL
  SELECT @RequestedServiceName WHERE @RequestedServiceName IS NOT NULL
),
ModeFiltered AS (
  -- Apply user override modes
  SELECT DISTINCT
    CASE
      WHEN @SelectionMode='VOLUME' THEN v.ServiceName
      WHEN @SelectionMode='INTENSITY' THEN i.ServiceName
      WHEN @SelectionMode='ALL' THEN s.ServiceName
      ELSE ss.ServiceName
    END AS ServiceName
  FROM SeedSelected ss
  LEFT JOIN Rank_Volume v ON @SelectionMode='VOLUME' AND ss.ServiceName = v.ServiceName AND v.rk <= @MaxOtherServices
  LEFT JOIN Rank_Intensity i ON @SelectionMode='INTENSITY' AND ss.ServiceName = i.ServiceName AND i.rk <= @MaxOtherServices
  LEFT JOIN PerServiceAgg s ON @SelectionMode='ALL' AND ss.ServiceName = s.ServiceName
),
RankedSelected AS (
  SELECT
    m.ServiceName,
    sc.ImpactScore,
    sc.ListAppearances,
    ROW_NUMBER() OVER (
      ORDER BY
        CASE WHEN m.ServiceName = @RequestedServiceName THEN 0 ELSE 1 END,
        sc.ListAppearances DESC,
        sc.ImpactScore DESC,
        m.ServiceName
    ) AS FinalRank
  FROM (SELECT DISTINCT ServiceName FROM ModeFiltered) m
  JOIN Scores sc ON sc.ServiceName = m.ServiceName
),
Limited AS (
  SELECT ServiceName
  FROM RankedSelected
  WHERE (@SelectionMode='ALL') OR FinalRank <= @MaxOtherServices OR ServiceName = @RequestedServiceName
),
HourlyFiltered AS (
  SELECT hb.*
  FROM HourlyBase hb
  JOIN Limited l ON l.ServiceName = hb.ServiceName
  WHERE @IncludeZeros = 1
     OR hb.ImpactedResource_Hour > 0
     OR hb.ServiceName = @RequestedServiceName
)
SELECT
  ServiceName,
  Offering,
  CalendarDayHour AS CalendarDayHour,
  MAX(AR_Fleet_Hour) AS ActiveResourceFleetCount,
  SUM(ImpactedResource_Hour) AS ImpactedResourceCount,
  CAST(ROUND(1.0*SUM(ImpactedResource_Hour)/NULLIF(MAX(AR_Fleet_Hour),0),4) AS DECIMAL(10,4)) AS AIRO_Hourly
  -- If AIROD requested:
  -- ,CAST(ROUND(1.0*SUM(ImpactedResource_Hour*ImpactDuration_Hour)/NULLIF(MAX(AR_Fleet_Hour),0),4) AS DECIMAL(10,4)) AS AIROD_Hourly
FROM HourlyFiltered
GROUP BY ServiceName, Offering, CalendarDayHour
ORDER BY
  CASE WHEN ServiceName = @RequestedServiceName THEN 0 ELSE 1 END,
  ServiceName,
  CalendarDayHour;
